# TESLA-and-GameStop-stock-analysis
# Extracting and visualizing stock and revenue data for Tesla and GameStop using Python, yfinance, web scraping, pandas, and Plotly.

Stock Analysis Dashboard: Tesla & GameStop
Project Overview

This project analyzes historical stock prices and revenue data for Tesla (TSLA) and GameStop (GME). Data was collected using the yfinance library and web scraping techniques with BeautifulSoup and pandas.

The project demonstrates:

Financial data extraction using yfinance
Web scraping HTML tables
Data cleaning and preprocessing
Revenue analysis
Interactive stock and revenue visualization
Dashboard creation using Plotly

Technologies Used:

Python
Pandas
NumPy
yfinance
Requests
BeautifulSoup
Plotly

Project Tasks:

Tesla Analysis
Extract Tesla stock data
Scrape Tesla quarterly revenue data
Clean and process revenue information
Visualize stock price vs revenue

GameStop Analysis:

Extract GameStop stock data
Scrape GameStop quarterly revenue data
Clean and process revenue information
Visualize stock price vs revenue
